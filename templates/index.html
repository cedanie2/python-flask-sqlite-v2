Hello Class! This is your First Web App using Python Flask!

Please read the following information:

The entire website is built using the Python Flask framework. The Flask app needs an app.py with all the Python code. Then, it needs a directory (folder) called templates. You will put all of your .html files in the templates folder. You will place all of your CSS, JavaScript, and Images in a folder called static. Inside of the static folder will be 3 additional subfolders called css, js, and img. This is where you will place your respective CSS, JavaScript, and Images. Additionally, this app is using SQLite as a database. SQLite is a simple open source light weight database built for small applications. SQLite requires that you have a .db file in your project. The .db file is essentially the database.
